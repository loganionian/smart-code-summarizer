# smart-code-summarizer

An AI tool that automatically generates concise summaries and documentation for codebases.

## Problem

Many developers struggle to understand large codebases quickly; this tool provides instant, clear summaries.

## Features

- Auto-generation of function/method summaries.
- High-level overviews of entire modules or files.
- Integration with popular IDEs for on-the-fly documentation generation.

## Tech Stack

Python, OpenAI API
